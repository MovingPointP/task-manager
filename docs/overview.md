# サービス概要

## 目的と責務

タスクの作成・完了管理・削除を行うシンプルなタスク管理 Web アプリケーション。

- タスクの追加（タイトル入力）
- タスクの完了/未完了の切り替え
- タスクの削除
- タスク一覧の表示（新しい順）
