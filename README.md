# タスク管理アプリ

## アプリの機能

- タスクの一覧表示
- タスクの追加
- タスクの完了/未完了の切り替え
- タスクの削除
- タスクのフィルタリング

## 使用技術スタック

| 技術 | 用途 |
|------|------|
| [Next.js 14](https://nextjs.org/) | フレームワーク（App Router） |
| [TypeScript](https://www.typescriptlang.org/) | 開発言語 |
| [Tailwind CSS](https://tailwindcss.com/) | スタイリング |
| [Prisma](https://www.prisma.io/) | ORM（データベース操作） |
| [Neon](https://neon.tech/) | PostgreSQL（無料DBホスティング） |
| [Vercel](https://vercel.com/) | デプロイ（無料Webホスティング） |
